validateState(path, embedded, version)

Source:
  • validate.js, line 17

Validate the HDF5 state file containing the analysis parameters and results. It is assumed that the HDF5 state file has already been extracted from the *.kana file, e.g., using bakana's parseKanaFile function.

Parameters:
Name Type Description
path string

Path to the HDF5 file. On browsers, this should reside in the virtual filesystem.
embedded boolean

Whether the input files were embedded or linked into the file.
version number

Version of the kana file, as an XXXYYYZZZ integer for version XXX.YYY.ZZZ, e.g., 1001000 for version 1.1.0.
Returns:

An error is raised if the state file is invalid.
